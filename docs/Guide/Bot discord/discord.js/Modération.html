Modération

Commandes de Modération​

Dans cette section, nous couvrirons les commandes de modération qui vous permettent de gérer les membres et les messages sur votre serveur Discord. Ces commandes sont conçues pour aider les modérateurs à maintenir un environnement positif et respectueux.

Expulser (Kick)​

if (command === 'kick') {
    // Vérifier les permissions de l'utilisateur
    if (!message.member.permissions.has('KICK_MEMBERS')) {
        await message.reply("Vous n'avez pas la permission d'expulser des membres.");
        return;
    }

    const member = message.mentions.members.first();
    if (!member) {
        await message.reply("Veuillez mentionner le membre à expulser.");
        return;
    }

    await member.kick();
    await message.reply(`${member.user.tag} a été expulsé.`);
}

Explication : Cette commande permet aux modérateurs d'expulser un membre du serveur. Ils doivent avoir la permission "Expulser des membres" pour utiliser cette commande. En tapant !kick @membre, le bot expulsera le membre mentionné et enverra un message de confirmation.

Bannir (Ban)​

if (command === 'ban') {
    // Vérifier les permissions de l'utilisateur

}

Explication : Cette commande permet aux modérateurs de bannir un membre du serveur. Ils doivent avoir la permission "Bannir des membres" pour utiliser cette commande. En tapant !ban @membre, le bot bannira le membre mentionné et enverra un message de confirmation.

Rendre muet et Désactiver le mode muet​

if (command === 'mute') {
    // Vérifier les permissions de l'utilisateur
    if (!message.member.permissions.has('MANAGE_ROLES')) {
        await message.reply("Vous n'avez pas la permission de gérer les rôles.");
        return;
    }

    const member = message.mentions.members.first();
    if (!member) {
        await message.reply("Veuillez mentionner le membre à rendre muet.");
        return;
    }

    let muteRole = message.guild.roles.cache.find(role => role.name === "Muted");
    if (!muteRole) {
        try {
            muteRole = await message.guild.roles.create({
                name: "Muted",
                color: "#000000",
                permissions: []
            });
            message.guild.channels.cache.forEach(async (channel) => {
                await channel.createOverwrite(muteRole, {
                    SEND_MESSAGES: false,
                    ADD_REACTIONS: false
                });
            });
        } catch (error) {
            console.log(error);
        }
    }

    member.roles.add(muteRole);
    await message.reply(`${member.user.tag} a été rendu muet.`);
}

if (command === 'unmute') {
    // Vérifier les permissions de l'utilisateur
    if (!message.member.permissions.has('MANAGE_ROLES')) {
        await message.reply("Vous n'avez pas la permission de gérer les rôles.");
        return;
    }

    const member = message.mentions.members.first();
    if (!member) {
        await message.reply("Veuillez mentionner le membre à désactiver le mode muet.");
        return;
    }

    let muteRole = message.guild.roles.cache.find(role => role.name === "Muted");
    if (!muteRole) {
        await message.reply("Le rôle 'Muted' n'a pas été trouvé.");
        return;
    }

    member.roles.remove(muteRole);
    await message.reply(`${member.user.tag} n'est plus en mode muet.`);
}

Explication : Ces commandes permettent aux modérateurs de rendre muet ou de désactiver le mode muet pour un membre. Lorsque la commande mute est utilisée, le bot vérifie d'abord si le rôle "Muted" existe. Si le rôle n'existe pas, il le crée avec les autorisations appropriées pour empêcher le membre de parler. Ensuite, le bot attribue ce rôle au membre mentionné.

Lorsque la commande unmute est utilisée, le bot vérifie également si le rôle "Muted" existe. S'il n'existe pas, le bot informe que le rôle n'a pas été trouvé. Si le rôle existe, le bot retire ce rôle du membre mentionné pour lui permettre de parler à nouveau.

Ces commandes sont utiles pour gérer les comportements indésirables et maintenir la discipline dans le serveur en contrôlant la capacité des membres à communiquer dans les salons.

Supprimer des messages​

if (command === 'clear') {
    // Vérifier les permissions de l'utilisateur
    if (!message.member.permissions.has('MANAGE_MESSAGES')) {
        await message.reply("Vous n'avez pas la permission de gérer les messages.");
        return;
    }

    const amount = parseInt(args[0]);
    if (isNaN(amount) || amount < 1 || amount > 100) {
        await message.reply("Veuillez spécifier un nombre de messages à supprimer entre 1 et 100.");
        return;
    }

    await message.channel.bulkDelete(amount + 1);
    await message.reply(`${amount} messages ont été supprimés.`);
}

Explication : Cette commande permet aux modérateurs de supprimer un certain nombre de messages dans un canal. Ils doivent avoir la permission "Gérer les messages" pour utiliser cette commande. En tapant !clear [nombre], le bot supprimera les messages et enverra un message de confirmation.

Ces commandes de modération sont essentielles pour maintenir un environnement convivial et bien géré sur votre serveur Discord. Elles offrent aux modérateurs les outils nécessaires pour réagir aux situations appropriées et pour maintenir un espace positif pour tous les membres.
